# My Content

This is where your generated content lives.

When you run the content skills, they save output here:

- `content-strategy.md` — Your content strategy (from `/content-foundation`)
- `style-profile.md` — Your blended voice profile (from `/creator-analysis`)
- `creator-analysis/` — Individual creator analysis docs
- `youtube-scripts/` — Full YouTube scripts
- `repurposed/` — Multi-platform content suites

This folder is gitignored so your personal content stays private.
